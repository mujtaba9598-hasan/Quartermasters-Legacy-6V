# Quartermasters — Strategic Consulting & Advisory

> California-based, globally connected. Premium consulting across six integrated verticals.

## Tech Stack
- **Framework**: Next.js 16.1.6 (App Router), React 19, TypeScript
- **Styling**: Tailwind CSS v4, Framer Motion 12, Lenis smooth scroll
- **3D**: Three.js / React Three Fiber / Drei
- **AI**: Claude API (Anthropic), Vercel AI SDK for streaming
- **Backend**: Supabase + pgvector (RAG), Redis (rate limiting + cache)
- **Design System**: "Sovereign Nexus" — Deep Harbor + Burnt Copper (#C15A2C)

## Services (6 Verticals)
1. Financial Advisory & Banking Services
2. Human Capital & Resource Management
3. Strategic Management Consulting
4. Technology & Innovation (R&D)
5. Organization & Event Management
6. IT Services (Software Dev & Web Apps)

## Pages (21)
- Homepage, About, Contact, Services hub
- 6 service detail pages
- Knowledge Base (index + category pages)
- Terms of Service, Privacy Policy
- Portal, 404

## Key Features
- **Q** — AI consultant (Claude-powered, streaming via Vercel AI SDK)
- **Q Proactive Welcome** — Typewriter greeting, 6 suggestion chips, FAB tooltip nudge
- **Iron Grip Pricing** — Deterministic state machine, server-enforced
- **Velvet Rope UI** — Standard vs Premium side-by-side pricing
- **ScrollStack** — Lenis-powered scroll-pinning card stack (react-bits)
- **3D Globe** — R3F globe with HQ pin (California) + connection beams
- **GlareHover** — Glassmorphism hover effects on service cards
- **ClickSpark** — Copper spark animation on CTA buttons
- **Cookie Consent** — GDPR/CCPA/PDPL geo-adaptive banner
